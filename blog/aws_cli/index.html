AWS Command Line Tool

By Yachi 01 January 0001

本篇內容分為 1. 什麼是AWS CLI? 2.AWS CLI的安裝、登入設定

1. 什麼是AWS CLI?

AWS Command Line Interface是管理 AWS 服務的統一工具， 透過AWS CLI ，我們可以從命令列控制AWS的服務，並利用指令碼將服務自動化。

2.AWS CLI的安裝、登入設定

AWC CLI的安裝可參考此連結，安裝完成後，我們可透過CMD(命令提示字元視窗)，使用 aws –version 進行安裝完成的確認。

接著，我們需要進行登入設定

  • Step1: 首先需要到 AWS IAM 介面中找到使用者對應的Access Key ID和Secret Access Keys

  • Step2: 之後，我們在CMD中輸入aws configure指令，將Access Key ID和Secret Access Keys填入

  • Step3: 設定完後，可透過aws configure list命令查當前環境設定：
