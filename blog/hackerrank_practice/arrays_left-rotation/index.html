HackerRank練習 Arrays: Left Rotation

By Yachi 25 February 2020

本文主要是用來紀錄Arrays: Left Rotation該題的解題想法

Part 1. 題目描述:

本題主要的題目描述可以參考此連結，簡單來說就是針對一個陣列，重複執行往左移的動作。例如將陣列[1,2,3,4,5]做兩次往左移後就會變成是[3,4,5,1,2]。

Part 2. 分析

我們可以將往左移的步驟拆開來看，首先第一次將陣列往左移時

原先的陣列為

[1,2,3,4,5]

第一次往左移後會變成

[2,3,4,5,1]

第二次往左移後會變成

[3,4,5,1,2]

仔細觀察會發現，每一次往左移時，會把在陣列最左邊的數字擺放置陣列的最右邊，而剩餘在陣列內的數字則會跟著往左移動

Part 3. 程式碼

因此，依照上述邏輯，撰寫的程式碼如下:

def rotLeft(a, d):

    for i in range(d):  # 移動d次

        temp = a[0]  # 每一次先將陣列最左邊的數字取出來

        del a[0]  # 接著，刪除最左邊的數字，讓陣列中只保留剩下的數字

        a.append(temp)  # 在剩餘的陣列中，加入一開始取出來的最左邊數字

    return a
